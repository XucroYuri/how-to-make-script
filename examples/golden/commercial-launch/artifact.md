# Commercial Launch Commercial Script

## Core Message
- Audience pain: 夏天通勤和排队时，热得立刻失去耐心。
- Product proof: 便携风扇主打“30 秒极速降温”，一开就能看到体感变化。
- Memory point: 小体积、随手带、热到崩溃时马上救场。

## Shot Flow
- 0-2s: 女生在烈日下排队，额头出汗，表情明显烦躁。
- 2-5s: 她按下风扇开关，头发和汗珠马上有变化，字幕打出“30 秒极速降温”。
- 5-8s: 切地铁站、户外通勤、等咖啡三个快节奏场景，连续证明便携性。
- 8-12s: 特写产品收纳进小包，朋友伸手借用，强化“随手带着走”的记忆点。
- 12-15s: 品牌名和产品名留屏，人物对镜头抬手示意“别再硬扛高温了”。

## CTA
- Action: 现在下单，带走你的随身降温位。
