# Interactive Quest Interactive Branch Map

## Player Goal
- 在药物极度稀缺的夜晚保住营地，同时决定自己是否愿意承担救宿敌的政治代价。

## State Variables
- `camp_trust`: 营地成员对玩家判断的信任度。
- `rival_debt`: 宿敌获救后对玩家欠下的人情或反噬概率。
- `medicine_stock`: 当前夜晚和后续任务可动用的药物余量。

## Branches
- Branch A: 救宿敌，立即失去部分营地支持，但换来后续关键情报和潜在联盟。
- Branch B: 把药留给营地核心成员，短期防线更稳，却让宿敌阵营把这次选择记成血债。
- Branch C: 暂不救任何一方，冒险夜出寻找替代药物，开启额外任务链并承担更高即时失败率。

## Convergence Plan
- 三条分支都会在次日黎明回到营地防御战。
- 差异不在主线是否继续，而在谁愿意并肩作战、营地库存还剩多少、以及宿敌线会以援手、报复或缺席哪种方式回收。
