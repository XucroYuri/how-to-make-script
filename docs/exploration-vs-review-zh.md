# 探索与评审

创作工作分成两个必须保持独立的阶段：

1. **探索** — 拓宽选项空间，大胆尝试，暂停软性合规检查。
2. **评审** — 对存活下来的方案进行把关、筛选和验证，对照真实约束条件。

这是一条排序规则，不是一张通行证。探索不会暂停硬性安全边界。评审不应该在探索开始之前就把它扼杀。
