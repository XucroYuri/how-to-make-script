# Roadmap

## Phase 1
- Bootstrap repository metadata and community files.
- Freeze taxonomy, routing, ID policy, and schema contracts.
- Add the root skill and the first wave of thin sub-skills.

## Phase 2
- Expand foundational knowledge atoms across story, scene, dialogue, pacing, and rewrite craft.
- Cover the first three focus clusters deeply:
  - narrative screenwriting
  - commercial / branded scripting
  - interactive / branching narrative
- Add real-world grounding packs for audience demand, commissioning fit, writer development, platform logic, and historical shifts.
- Harden the dual-track editorial system so English stays agent-facing while Chinese stays practitioner-facing.
- Use the Socratic question backlog as the default intake gate before adding or splitting knowledge assets.

## Phase 3
- Add more genre packs and medium-specific constraints.
- Expand case studies and golden fixtures.
- Increase fixture depth for audience-fit, development strategy, and writer training routes.
- Add richer lint rules for contradictory or underspecified asset links.

## Phase 4
- Introduce optional generated indexes and registry publishing.
- Add translation layers and multi-language knowledge views.
- Add advanced rubric-driven rewrite loops for downstream agent systems.
