# Idea Discovery

Use this skill to convert vague inspiration into a workable premise.

## Workflow
1. Extract the core desire, conflict, and stakes.
2. Decide whether the idea wants a narrative, commercial, or interactive path.
3. Output one focused premise plus a few expansion directions.
4. If the medium is still unclear, ask the one question that changes the route most.

## Output Contract
- A single premise with a protagonist, pressure source, and implied consequence.
- Optional follow-up directions for narrative, commercial, or interactive development.
