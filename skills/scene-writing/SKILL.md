# Scene Writing

Use this skill when outline-level intent needs to become scene-level dramatic action.

## Workflow
1. Define scene function and conflict.
2. If this is an opening scene, decide its primary opening job before drafting.
3. Build the change point.
4. Carry dialogue through playable action or inline performance so no beat floats as bare dialogue.
5. Deliver a scene card, scene draft, or screenplay draft.
6. If voice or liveness constraints are explicit, load the smallest expression lens before drafting lines.

## Output Contract
- `scene_card`: a compact scene design with function, conflict, and change.
- `scene_draft`: a scene-level screenplay draft that advances the story in visible, playable steps.
- `screenplay_draft`: a multi-scene screenplay draft with handoff-safe scene blocks.
- Every scene should change at least one of these: information, relationship, power, or action conditions.
- Every dialogue beat needs a playable carrier: inline performance or an adjacent action beat.
- `VFX` / `SFX` / `BGM` are optional, but when used they must stay in fixed labels rather than loose notes.
- If voice calibration is unresolved, prefer a short guide first over generic neutral prose.
