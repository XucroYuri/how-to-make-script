# Commercial Script

Use this skill when the script must sell, persuade, or create a branded memory.

## Workflow
1. Lock the single core message.
2. Map hook, proof, memory point, and CTA.
3. Deliver a concise commercial script matched to platform and duration.
4. If the brief hinges on tone, define the brand register before writing copy.

## Output Contract
- `commercial_script`: a short, conversion-aware script for ad, branded film, or shortform use.
- Keep the message singular unless the brief explicitly calls for a message stack.
- Optimize for recall, platform fit, and a believable action prompt.
- Premium or trust-heavy tone should be grounded in lived moments, not luxury filler language.

## Posture-Adaptive Guidance

**When `certainty = lost`:**
Ask: "What is the one thing you want the viewer to feel — not think, *feel* — in the last three seconds?" A single emotional target unlocks the whole brief. Do not present a framework.

**When `source = discover`:**
Surface the brand's genuine tension: what does it actually believe vs. what the brief claims it believes? The sharper that gap, the stronger the script premise.

**When `source = construct`:**
Lock message first (single), then sequence: hook → proof → memory point → CTA. Platform fit and duration are constraints, not afterthoughts.

**When `focus = audience`:**
Audience payoff and platform retention window are the primary load. A commercial that satisfies the brief but loses the viewer in the first three seconds has failed both jobs.

**When `focus = language`:**
Register control matters more in commercial work than in any other medium. Premium tone must be grounded in lived moments; aspirational language without specificity reads as filler.
