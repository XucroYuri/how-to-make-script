# Writer Development

Use this skill to turn broad learning goals into a disciplined screenwriter learning path.

## Workflow
1. Locate the writer's current maturity and recurring failure layer.
2. Define target capability relative to medium, genre, and production context.
3. Sequence learning, writing, feedback, and rewrite loops.
4. Return one practical learning path with milestones.

## Output Contract
- `learning_path`: a staged path with capability goals, practice loops, and measurable checkpoints.
- Include feedback channels and revision cadence, not just reading lists.
- Keep milestones connected to real writing output artifacts.
