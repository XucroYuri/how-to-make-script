# Rewrite Report Rubric

用于判断改稿报告是不是能真正指导下一轮重写。

改稿报告的价值，不在于它指出了多少毛病，而在于它有没有帮助团队决定“下一轮最该先改什么”。这个 rubric 会特别强调分层、排序和可执行性，因为没有这三件事的诊断通常只会制造更多讨论，不会制造更好的稿子。
