# Dialogue Rubric

用于判断对白是否有戏，而不仅仅是有字。

对白是最容易被“朗朗上口”误导的层。这个 rubric 会强迫你回头看更关键的东西：人物声音有没有差异，潜台词有没有在运作，信息是不是被过度解释。真正有戏的对白，不靠金句撑场，而靠关系和策略发力。

## 示例对照

**Hard fail**（声音坍塌——所有角色同一口吻）：
A: 我觉得我们应该好好谈谈这件事。
B: 我也认为我们需要认真沟通一下。
C: 确实，大家应该坦诚地交流各自的看法。
→ 三人句式相同、语域相同、没有角色区分。

**Workable**（功能正常但扁平）：
A: 你为什么不告诉我？
B: 我以为你不关心。
A: 你怎么能替我决定？
→ 有基本冲突轮次，但全是直接情绪陈述，没有潜台词。

**Excellent**（声音 + 潜台词 + 节奏）：
A: 那个盒子呢。
B: 什么盒子。
A: 你知道我说哪个。
B: ……我扔了。
A: 不，你没有。
→ 短/逃避节奏来自 B，压迫节奏来自 A；潜台词指向两人都知道但不命名的东西；没有人直接说出情绪。
