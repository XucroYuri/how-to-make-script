# Case Study: Interactive Quest

Use the interactive quest golden example to study branch meaning, state tracking, and convergence planning.

这个案例最值得看的不是分了三条路，而是三条路分别改变了什么：价值判断、资源后果还是关系状态。做互动项目时，案例真正提供的不是“树状图模板”，而是“如何让选择留下痕迹”的思路。
