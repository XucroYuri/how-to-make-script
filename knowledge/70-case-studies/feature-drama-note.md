# Case Study: Feature Drama

Use the feature drama golden example to study how a premise grows into beat sheet, outline, and scene-level choices without losing the protagonist goal.

中文阅读时，可以重点观察三件事：主角目标在不同开发层级有没有跑偏；中段升级是不是由前面选择逼出来的；关键场景有没有真正承担结构功能。长片案例最适合拿来练“如何避免中段空转”。
